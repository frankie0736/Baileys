# User Stories

## 核心场景

| 角色 | 目标 | 价值 |
|------|------|------|
| 用户 | 发文字消息给机器人，得到智能回复 | 解答疑问 |
| 用户 | 连续发多条消息，机器人理解整体意图 | 不用等每条都回 |
| 用户 | 长回复分多条发送，像真人打字 | 自然交互体验 |
| 用户 | 机器人记住之前聊了什么 | 连贯对话 |

## 边界场景

| 场景 | 期望行为 |
|------|----------|
| AI 不知道答案 | 说"不确定" + 引导换个问法 |
| 用户发图片/视频/文件 | 保持现有行为（收到+保存） |
| AI 回复很长 | 智能分段，逐条发送 |
| 用户 5 秒内连发多条 | 合并后一起理解 |
